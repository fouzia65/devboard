Question-1: What are the different ways to select an element in the DOM?

বিভিন্ন ভাবে DOM হতে উপাদান নির্বাচন করা যায় সেগুলো হল ঃgetElementsByName(),querySelectorAll(),querySelector() ,getElementsByTagName() ,getElementsByClassName() etc

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHtml: HTML ট্যাগসহ ভিতরের লেখা গুলো পাওয়ার জন্য ব্যবহার করা হয় innerText: ট্যাগ বাদে শুধু ট্যাগ এর ভেতরের লেখা গুলা পাওয়ার জন্য ব্যাবহার করা হয় textContent : উপস্থিত অনুপস্থিত সকল টেক্সট পাওয়ার জন্য ব্যাবহার করা হয়

Question-3: What is event delegation in the DOM?

event delegation হল এমন একটি প্রযুক্তি যা child element এর জন্য আলাদা আলাদা addEvenListener() ব্যাবহার না করে child এর parent element এ যোগ করা হয়

Question-4: What is event bubbling in the DOM?

event delegation হল এমন একটি প্রযুক্তি যা child element এর জন্য আলাদা আলাদা addEvenListener() ব্যাবহার না করে child এর parent element এ যোগ করা হয়। parent element child element এর জন্য addEventListener() এর কাজটি করে একে event bubbling বলে

Question-5: How do you create, add, and remove elements using JavaScript?

document.createElement() ব্যাবহার করে উপাদান create করতে পারি। উপাদান add করার জন্য appendChild() ব্যাবহার করতে পারি । removeChild() or remove() ব্যাবহার করে উপাদান remove করতে পারি
